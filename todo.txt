Phase 1: Security Hardening (CRITICAL)
 Generate and set SECRET_KEY in environment variable
 Move all API URLs to environment variables
 Configure CORS with specific domain
 Set up HTTPS/TLS with Let's Encrypt
 Add rate limiting to auth endpoints
 Review and sanitize all user inputs
 Set proper file permissions (600 for .db, 644 for code)

Phase 2: Infrastructure Setup
 Set up VPS (recommended: Hetzner CX21 or DigitalOcean Basic Droplet)
 Install Docker and Docker Compose
 Configure nginx as reverse proxy
 Set up automatic SSL renewal
 Configure firewall (UFW): allow 80, 443, 22 only
 Set up automated backups for database

Phase 3: Application Deployment
 Create production Dockerfile for backend
 Create production Dockerfile for frontend (nginx serve)
 Set up docker-compose.yml with proper networking
 Configure environment variables via .env file
 Implement health check endpoints
 Set up logging (structured JSON logs)
 Configure log rotation
 
Phase 4: Monitoring & Maintenance
 Set up monitoring (UptimeRobot, Grafana, or similar)
 Configure error tracking (Sentry recommended)
 Set up automated database backups (daily)
 Create backup restoration procedure
 Document deployment process
 Set up CI/CD pipeline (GitHub Actions)
